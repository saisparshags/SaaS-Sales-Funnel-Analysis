SaaS Sales Funnel Analysis Dashboard
This project analyzes a SaaS sales funnel using a Power BI dashboard to understand customer conversion and identify bottlenecks in the sales process. It tracks the journey from Leads to Closed Deals, including key stages like MQL, SQL, and Demo.

🛠️ Tools Used
Power BI
Microsoft Excel

🔍 Key Insights
* Significant drop from SQL → Demo stage
* Customer Acquisition Cost (CAC) is relatively high
* Leads are increasing, but revenue growth is not proportional

💡 Recommendations
* Improve demo conversion through better onboarding
* Optimize marketing channels to reduce CAC
* Focus on high-value customer segments
